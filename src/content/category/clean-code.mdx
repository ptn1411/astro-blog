---
name: Clean code
description: Clean code, hay "mã sạch", là mã nguồn được viết theo các nguyên tắc và quy ước để dễ đọc, dễ hiểu, dễ bảo trì và dễ thay đổi. Mục tiêu chính là cải thiện chất lượng phần mềm, giúp các lập trình viên khác và chính bạn có thể làm việc hiệu quả hơn với mã nguồn trong tương lai.
