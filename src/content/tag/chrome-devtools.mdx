---
name: Chrome DevTools
description: Chrome DevTools là một bộ công cụ dành cho nhà phát triển web được tích hợp sẵn trong trình duyệt Google Chrome. Nó cho phép bạn xem, chỉnh sửa và gỡ lỗi mã HTML, CSS và JavaScript, giúp bạn xây dựng và tối ưu hóa trang web nhanh hơn. Bạn có thể truy cập DevTools bằng cách nhấn F12 hoặc Ctrl + Shift + I (trên Windows/Linux) hoặc Cmd + Opt + I (trên macOS).
